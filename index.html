Welcome to WMP Team A
Web Map Project page

These maps are created for the Web Map Project course in Aalto University. They depict service requests given to the City of Helsinki. City residents can check the status of their feedback on a map and observe what kind of service requests and feedbacks are given for example in their neighbourhood or in other areas of interest.

Data for the maps is from City of Helsinki’s Open 311 API which is an API for reporting issues and giving feedback.

Web map using Google Maps API

The web maps are implemented using two different techniques: Google Maps API and OpenLayers 3.

Google Maps version has custom and basic background maps to choose from. Check the map here.
